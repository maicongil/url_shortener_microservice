URL Shortener Microservice

  • User Story: I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • User Story: If I pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.
  • User Story: When I visit that shortened URL, it will redirect me to my original link.

Usage:

To generate a shortened url, send a request to "https://fccshortit.herokuapp.com/new/[url]", where [url] is the url to be shortened

To use the shortened url and get redirected to original url, send a request to "https://fccshortit.herokuapp.com/[shortenedUrl]", where [shortenedUrl] is your shortened url

Example:

Input: https://fccshortit.herokuapp.com/new/http://www.twitter.com

Output :{"originalUrl":"http://www.twitter.com","shortenedUrl":"4567"}

Acessing "https://fccshortit.herokuapp.com/4567" you will be redirected to "http://www.twitter.com"
